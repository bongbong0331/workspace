JSP & Servlet - MySQL

JDBC 란 ? 
JAVA 와 MySQL( DBMS ) 이 통신할 수 있는 방법을 제시한다.

JDBC 란 ?
JSP 를 비롯한 Java 기반의 어플리케이션 데이터베이스를 사용할 때 JDBC API 를 사용하여 코딩을 한다.
각각의 DBMS 는 자신에게 적절한 JDBC 드라이버를 제공하며, 
JDBC API는 JDBC 드라이버를 거쳐서 데이터 베이스와 통신을 합니다.
즉, DBMS에 알맞은 JDBC 드라이버가 존재 한다면 어떤 데이터 베이스라도 사용할 수 있습니다.
그렇기 때문에 DBMS에 알맞은 JDBC 드라이버가 있어야 하고, 
이 JDBC 드라이버는 클래스 형태로 존재하며 일반적으로 .jar 파일로 제공 됩니다
